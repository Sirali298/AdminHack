* `pkg update`
* `pkg install git`

## Installing for linux

* `apt update`
* `apt upgrade`
* `apt install git`
* `git clone https://github.com/sadamshr3be/AdminHack`
* `cd AdminHack`
* ` unzip src.zip`
* `bash setup.sh`
